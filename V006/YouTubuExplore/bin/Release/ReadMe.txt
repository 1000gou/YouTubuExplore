≪ソフトウェアの概要≫
YouTubeの関連動画のつながり示す関係図を動画のサムネイルを使って表示するアプリケーションです。
動画のサムネイルを左クリックするとで関連動画を追っていくことができ、右クリックすることでWebブラウザで再生できます。

≪インストール方法、実行方法、アンインストール方法≫
1.このアプリケーションを実行するためには 『Microsoft .NET Framework 4 』がインストールされている必要があります。
 『Microsoft .NET Framework 4 』は2011/05/02 時点では以下のマイクロソフトのサイトからインストールできます。
http://www.microsoft.com/downloads/details.aspx?familyid=9CFB2D51-5FF4-4491-B0E5-B386F32C0992&displaylang=ja　

2.YouTubeExplorerのインストールは必要ありません。
　YouTubeExplorer_v004.exeをダブルクリックして実行してください。

3.YouTubeExplorerのアンインストール方法
　YouTubeExExplorerのフォルダを手動で削除してください。

≪操作方法≫
●初期ページの読み込み
テキストボックスへ『YouTubeのURLたとえばhttp://www.youtube.com/』または『検索ワード』を入力し、読み込みボタンを押す。
⇒初期ページにあるサムネイルの画像がダウンロードされます。YouTubeExplorerではこの読み込まれたサムネイル画像をノードと読んでいます。

●サムネイル画像ページの読み込み
読み込んだサムネイルの画像を左ダブルクリックすると、サムネイル画像のページを読み込みそのページにあるサムネイル画像を読み込みます。

●サムネイル画像を左クリックするとタイトルを表示します。

●サムネイル画像ページをWebブラウザで再生
読み込んだサムネイルの画像を右クリックすると、サムネイル画像のページの動画をWebブラウザで再生します。

●サムネイル画像ページを左ドラッグすることで、画像を移動します。

●画像のない場所を左ドラッグすることで、表示する範囲を移動します。

●マウスのホイールでズームインおよびズームアウトが可能です。
